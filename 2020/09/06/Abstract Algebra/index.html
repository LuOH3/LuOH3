Examples from equations

firstly, we consider the polynomial equation

Rmk. in order to find $(x_1,\dots,x_n)$, we need to define the so-called “number” first, and it should be equipped with $+$ and $\cdot$

the most simple number set we can find is $N(N^\times)$

from the angle of equation, we find out that

(i) $x+a=b$ in $N(N^\times)$ might have no solution, thus, we consider $Z$, then $+$ is invertible
(ii) $ax=b$ in $Z$ might have no solution, thus, we consider $Q$, then $\cdot$ is invertible in $Q\setminus\lbrace 0\rbrace$
(iii) $x^2=2$ in $Q$ might have no solution, thus, we consider $R$, then the measurements in physics become meaningful
(iv) $x^2+1=0$ in $R$ might have no solution, thus, we consider $C$, which is an algebraic closed field, i.e. $\forall a_0,\dots,a_n\in C,a_0+a_1x+\dots+a_nx^n=0$ must have $n$ roots

we have more examples of the “number” set

(1) $H:=\lbrace a_0+a_1i+a_2j+a_3k|a_0,a_1,a_2,a_3\in R\rbrace$, where $i^2=j^2=k^2=-1,i\cdot j=k=-j\cdot i,j\cdot k=i=-k\cdot j,k\cdot i=j=-i\cdot k$

(2) $M_n(C)$ (note that $\cdot$ is not interchangable)

(3) let $S$ be a set, then $\text{Map}(S,R),C^0(S,R),C^n(S,R),C^\infty(S,R)$ are all equipped with $+$ and $\cdot$

Defs.

(1) binary operation

a binary operation $\circ$ on a set $S$ is a mapping $S\times S\to S:(a,b)\to a\circ b$

(2) field

A field is a set $K$ equipped with $+$ and $\cdot$ which satisfies

F1: $+:S\times S\to S:(a,b)\to a+b$ satisfies

F2: $\cdot:S\times S\to S:(a,b)\to a\cdot b$ satisfies

F3: $+$ and $\cdot$ satisfies

Rmk. we always ask that $0_K\neq 1_K$, otherwise $K=\lbrace 0_K\rbrace$, which is trivial

Rmk. to simplify, we often write $na,a^n(n\in Z)$

the examples of field are $Q,R,C$

(3) ring

a ring is a set $R$ equipped with $+$ and $\cdot$ which satisfies

R1=F1

R2=F2#1,#2

R3=F3

we define $R^\times:=\lbrace x\in R|x \text{ is invertible}\rbrace$

the examples of ring are $C^n(I,R),M_n(C)$

(4) group

a group is a set $G$ equipped with $\cdot$ which satisfies

G=F2#1,#2,#3

Rmk. $1_G$ in the group is always written as $e_G$

the examples of group are $(field,+),(field\setminus\lbrace 0\rbrace,\cdot),(ring,+)$

(5) monoid 幺半群

a monoid is a set $M$ equipped with $\cdot$ satisfies

M=F2#1,#2

the examples of monoid are $(N,+),(N,\cdot)$

similarly, we can define homomorphism between rings, groups and monoids

Examples from number theory

Congruence class 同余类

$n\equiv n’\mod m\Leftrightarrow m|n-n’$, defined as $n\stackrel{m}{\sim}n’$

then $\stackrel{m}{\sim}$ is a equivlance relation, which defines the equivlance class $Z/m=Z/mZ=\lbrace \overline{0},\overline{1},\dots,\overline{m-1}\rbrace$

define $\overline{n}+\overline{n’}=\overline{n+n’},\overline{n}\cdot\overline{n’}=\overline{n\cdot n’}$, then $Z/mZ$ is a ring, and note that $Z\to Z/mZ:a\to\overline{a}$ is a homomorphism on the ring

furthermore, if $m$ is a prime, $Z/mZ$ is a field

if $m=p^e,p$ is a prime, then $#(Z/mZ)^\times=p^e(1-\frac1p)$

generally, if $m=p_1^{e_1}…p_r^{e_r}$, then $Z/mZ\to Z/p_1^{e_1}Z\times\dots\times Z/p_r^{e_r}Z$ is a homomorphism on the ring, thus, $#Z/mZ=m\prod_{p|m}(1-\frac1p)$
